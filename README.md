Backend PHP estructurado para la base de datos "fernando_inmobiliaria".

Pasos rápidos:

1. Copiar `.env.example` a `.env` y configurar credenciales.
2. Ejecutar `composer install`.
3. Servir la carpeta `public` (por ejemplo `php -S localhost:8000 -t public`).

Notas:
- Usa JWT para autenticación.
- Repositorios y controladores están en `app/Repositories` y `app/Controllers`.
- Añadir validaciones adicionales según necesidades.
